# AI SQL Agent 🚀

AI-powered SQL assistant that converts natural language into SQL queries, validates them, and executes against a database with real-time results.

- 🔐 JWT Authentication
- 🧠 Natural Language → SQL
- 📊 Query Execution + Results Table
- 🕘 Query History with Re-run
- 📁 Database Tables Viewer
- ⚡ Real-time API integration
- 🎨 Modern UI with React + Tailwind + shadcn

Frontend:

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui

Backend:

- Node.js + Express
- PostgreSQL
- JWT Authentication
